# Claude Code 설치 가이드

> **워크샵 사전 준비** | 소요 시간: 15분
>
> 본인 컴퓨터가 **Mac**인지 **Windows**인지 확인 후, 해당 섹션만 따라하세요.

---

## GitHub 계정 만들기

> 팀 협업 및 코드 공유를 위해 필요합니다.

1. [github.com](https://github.com) 접속
2. **Sign up** 클릭
3. **Continue with Google** 클릭 → 본인 Google 계정으로 로그인
4. 이메일 인증 완료

> 💡 Google로 가입하면 비밀번호 관리가 편하고, Vercel 등 다른 개발 서비스 가입 시에도 유용합니다.

## Mac 유저

### 1. Claude 계정 만들기
1. [claude.ai](https://claude.ai) 접속
2. 계정 생성 후 **Pro 또는 Max** 구독

### 2. VS Code 설치
> 💡 이미 **Cursor** 또는 **Antigravity** 쓰시는 분은 이 단계 건너뛰세요!

1. [code.visualstudio.com](https://code.visualstudio.com/download) 접속
2. **Mac** 버전 다운로드
3. 다운로드된 파일 실행 → 응용 프로그램 폴더로 드래그

### 3. Claude Code 설치 (CLI)
1. VS Code 상단 메뉴에서 **Terminal** → **New Terminal** 클릭
   > 💡 단축키: `` Ctrl + ` `` (백틱, 숫자 1 왼쪽 키)
2. 아래 명령어 복사 → 터미널에 붙여넣기 → Enter
   ```
   curl -fsSL https://claude.ai/install.sh | bash
   ```
3. VS Code 완전히 종료 후 다시 열기
4. 터미널에서 확인: `claude --version` 입력 → 버전 나오면 성공

### 4. Claude Code 로그인 (CLI)
1. 터미널에 `claude` 입력 → Enter
2. 브라우저가 자동으로 열림 → Claude 계정으로 로그인
3. **허용** 클릭
4. 터미널로 돌아와서 `안녕하세요` 입력 → 응답 오면 **CLI 설정 완료!**

### 5. Claude Extension 설치 (VS Code)
1. VS Code 왼쪽 사이드바에서 **Extensions** 아이콘 클릭 (네모 4개 모양)
   > 💡 단축키: `Cmd + Shift + X`
2. 검색창에 **Claude** 입력
3. **Anthropic** 제작 확인 후 **Install** 클릭

### 6. Extension 로그인
1. VS Code 왼쪽 사이드바에서 **Claude 아이콘** 클릭 (방금 설치한 Extension)
2. **Sign In** 버튼 클릭
3. 브라우저에서 Claude 계정 로그인 → **허용** 클릭
4. Claude 패널에 `안녕하세요` 입력 → 응답 오면 **완료!**

## 문제 해결

| 증상 | 해결 |
|------|------|
| `claude` 명령어가 안 됨 | VS Code 완전히 종료 후 재시작 |
| 브라우저 로그인 후 터미널 반응 없음 | 브라우저에서 "허용" 클릭 확인 |
| Extension이 안 보임 | VS Code 재시작 |
| 로그인이 안 됨 | 브라우저 팝업 차단 해제 |
| 응답이 안 옴 | Claude 구독 상태 확인 |

**해결 안 되면**: 워크샵 당일 12:30에 도움드립니다.

## 워크샵 사전 스킬 설치 & 실행

> 🎯 **필수!** 워크샵 전에 이 스킬로 본인 업무를 정리하고, 만들 스킬 설계서를 받아오세요.

### 1. 사전 스킬 설치

터미널에서 아래 명령어 실행:
```
claude install github:next-company/cc-skill-workshop/skills/workshop-prep
```

> 💡 `claude install`은 GitHub에서 스킬을 다운받아 설치하는 명령어입니다.

### 2. 스킬 실행

터미널에서 `claude` 실행 후:
```
/workshop-prep
```

또는 자연스럽게:
```
워크샵 준비 도와줘
```

### 3. 대화 따라가기

스킬이 몇 가지 질문을 합니다:
1. 현재 하시는 일 (직무/역할)
2. 반복적으로 하는 작업
3. 가장 귀찮은/시간 걸리는 작업
4. 현재 프로세스 (시작 → 끝)
5. 이상적인 자동화 결과

**솔직하고 구체적으로** 답변해주세요!

### 4. 설계서 저장

대화가 끝나면 **스킬 설계서**가 생성됩니다.
이 설계서를 **워크샵 당일 가져오세요!**

저장 방법:
- 복사해서 노트앱에 붙여넣기
- 또는 파일로 저장 요청: `이거 파일로 저장해줘`
