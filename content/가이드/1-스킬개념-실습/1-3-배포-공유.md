## 왜 배포하나요?

| 로컬에만 있으면 | GitHub에 올리면 |
|----------------|----------------|
| 내 컴퓨터에서만 사용 | 어디서든 설치 가능 |
| 공유하려면 파일 전송 | 링크 하나로 공유 |
| 버전 관리 어려움 | 업데이트 추적 가능 |

> 💡 **비유**: 앱을 폰에만 갖고 있기 vs 앱스토어에 올리기

---

## 실습 1: GitHub에 배포하기

### Step 1: GitHub 저장소 만들기

1. [github.com](https://github.com) 접속
2. 오른쪽 위 **+** 버튼 → **New repository**
3. 설정:
   - **Repository name**: `my-skills` (또는 원하는 이름)
   - **Public** 선택 (다른 사람이 설치하려면 필수!)
   - **Add a README file** 체크
4. **Create repository** 클릭

---

### Step 2: 스킬 폴더 업로드

**방법 A: 드래그 앤 드롭 (가장 쉬움)**

1. 만든 저장소 페이지에서 **Add file** → **Upload files**
2. `.claude/skills/스킬이름` 폴더를 통째로 드래그
3. **Commit changes** 클릭

**방법 B: Claude에게 요청**

```
내 스킬을 GitHub에 올려줘.
저장소: https://github.com/내아이디/my-skills
```

---

### Step 3: 저장소 구조 확인

업로드 후 이런 구조가 되어야 합니다:

```
my-skills/
├── README.md
└── meeting-summary/          ← 스킬 폴더
    └── SKILL.md
```

> ⚠️ **주의**: `SKILL.md`가 폴더 안에 있어야 합니다. 루트에 바로 있으면 안 돼요!

---

## 실습 2: 옆 사람 스킬 설치하기

### Step 1: GitHub 주소 확인

옆 사람에게 물어보세요:
- "GitHub 저장소 주소가 뭐예요?"
- 예: `https://github.com/friend-id/my-skills`

---

### Step 2: 마켓플레이스 등록

```
/plugin marketplace add friend-id/my-skills
```

> 💡 `friend-id/my-skills` 부분을 옆 사람 저장소로 바꾸세요

---

### Step 3: 스킬 설치

```
/plugin install meeting-summary@friend-id/my-skills
```

> 💡 `meeting-summary` 부분을 옆 사람 스킬 이름으로 바꾸세요

---

### Step 4: 설치 확인 & 테스트

```
/skills
```

→ 옆 사람 스킬이 목록에 보이면 성공!

이제 그 스킬을 사용해보세요:
```
[옆 사람 스킬의 트리거 문구 입력]
```

## 💡 스킬 공유의 의미

### 팀 업무 자동화

```
1. 한 사람이 유용한 스킬 만듦
   ↓
2. GitHub에 배포
   ↓
3. 팀원 모두 설치
   ↓
4. 팀 전체 업무 효율 ↑
```

### 실제 활용 예시

| 역할 | 만들 수 있는 스킬 | 공유 대상 |
|------|------------------|----------|
| PM | 회의록 요약, PRD 템플릿 | 전 팀원 |
| 디자이너 | 피드백 정리, 에셋 네이밍 | 디자인팀 |
| 리서처 | 인터뷰 분석, 인사이트 추출 | 프로덕트팀 |

> 🎯 **핵심**: "내가 자동화한 게 팀 전체에 적용"

## 스킬 저장소 관리 팁

### 여러 스킬 한 저장소에

```
my-skills/
├── README.md
├── meeting-summary/
│   └── SKILL.md
├── feedback-organizer/
│   └── SKILL.md
└── prd-template/
    └── SKILL.md
```

### README.md 예시

```markdown
# 내 스킬 모음

## 설치 방법
/plugin marketplace add 내아이디/my-skills
/plugin install 스킬이름@내아이디/my-skills

## 스킬 목록
- meeting-summary: 회의록 요약
- feedback-organizer: 피드백 정리
- prd-template: PRD 작성
```

## ⚠️ 문제 해결

### "저장소를 못 찾아요"

1. 저장소가 **Public**인지 확인
2. 주소 오타 확인: `github.com/아이디/저장소`

### "스킬 설치가 안 돼요"

1. 마켓플레이스 등록 먼저 했는지 확인
2. 스킬 폴더 이름 정확히 입력
3. 폴더 구조 확인 (SKILL.md가 폴더 안에 있어야 함)

### "설치했는데 작동 안 해요"

1. `/skills`로 설치 확인
2. description에 맞는 키워드로 호출
3. Claude Code 재시작
