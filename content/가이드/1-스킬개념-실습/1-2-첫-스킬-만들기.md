# 1.2 첫 스킬 만들기

> skill-creator 스킬을 사용해서 나만의 첫 스킬을 만들어봅니다.

---

## 🎯 이번 시간 목표

- [ ] skill-creator로 스킬 생성하기
- [ ] 생성된 SKILL.md 확인하기
- [ ] 스킬 테스트해보기

---

## skill-creator란?

**스킬을 만드는 스킬**입니다.

| 직접 만들기 | skill-creator 사용 |
|------------|-------------------|
| SKILL.md 문법 공부 필요 | 대화하면서 자동 생성 |
| frontmatter 형식 외워야 함 | 질문에 답하면 완성 |
| 오타나면 에러 발생 | 검증된 구조로 생성 |

> 💡 **비유**: 웹사이트 만들 때 HTML 직접 쓰기 vs 노션/위즈/프리즘 사용하기

---

## 실습: 첫 스킬 만들기

### Step 1: skill-creator 실행

채팅창에 입력:

```
/skill-creator
```

Claude가 몇 가지 질문을 합니다. 아래처럼 답해보세요.

---

### Step 2: 질문에 답하기

**질문 1: "어떤 스킬을 만들고 싶으세요?"**

예시 답변:
```
회의록 요약 스킬을 만들고 싶어요.
긴 회의록을 주면 핵심 결정사항과 액션아이템만 뽑아주는 거예요.
```

**질문 2: "어떤 상황에서 이 스킬을 사용하나요?"**

예시 답변:
```
- 팀 회의 후 회의록 정리할 때
- 길게 적힌 미팅 노트 받았을 때
- "회의 요약해줘", "액션아이템 뽑아줘" 라고 말할 때
```

**질문 3: "결과물은 어떤 형태가 좋을까요?"**

예시 답변:
```
마크다운으로:
- 한 줄 요약
- 핵심 결정사항 (불릿)
- 액션아이템 (담당자, 마감일 포함)
```

---

### Step 3: 생성된 스킬 확인

Claude가 SKILL.md 파일을 만들어줍니다.

**확인할 것:**

1. **파일 위치**: `.claude/skills/meeting-summary/SKILL.md`
2. **frontmatter 확인**:
   ```yaml
   ---
   name: 회의록 요약
   description: 긴 회의록을 핵심 결정사항과 액션아이템으로 요약합니다. 
     "회의 요약해줘", "액션아이템 뽑아줘" 라고 말하면 자동 실행됩니다.
   ---
   ```
3. **본문**: 스킬이 따라야 할 절차가 적혀있음

---

### Step 4: 스킬 테스트

**테스트 1**: 스킬 목록에서 확인
```
/skills
```
→ 방금 만든 스킬이 보이면 성공!

**테스트 2**: 실제로 사용해보기
```
아래 회의록 요약해줘:

[오늘 회의]
참석: 김PM, 이디자이너, 박개발
- 신규 기능 A 12월 출시 확정
- 디자인은 다음주 금요일까지
- 개발 착수는 디자인 완료 후
- 박개발: API 문서 먼저 정리하기로 함
- 다음 회의는 수요일 2시
```

→ 스킬이 자동으로 실행되면 성공!

---

## 예상 결과물

```markdown
## 한 줄 요약
신규 기능 A 12월 출시 확정, 디자인 → 개발 순서로 진행

## 핵심 결정사항
- 신규 기능 A 12월 출시 확정
- 디자인 완료 후 개발 착수

## 액션아이템
| 담당자 | 할 일 | 마감 |
|--------|-------|------|
| 이디자이너 | 디자인 완료 | 다음주 금요일 |
| 박개발 | API 문서 정리 | - |

## 다음 회의
- 일시: 수요일 2시
```

## 💡 잘 만들어진 스킬의 특징

### description이 잘 써졌는가?

| 나쁜 예 | 좋은 예 |
|---------|---------|
| "회의록 요약" | "회의록을 요약합니다. '회의 요약해줘', '액션아이템 뽑아줘'라고 말하면 실행" |
| 한 단어 | **무엇을 하는지 + 언제 쓰는지** 모두 포함 |

> 🎯 **핵심**: description은 스킬의 **트리거 조건**입니다. 
> 사용자가 말할 법한 키워드를 꼭 넣으세요!

---

### 본문이 명확한가?

| 나쁜 예 | 좋은 예 |
|---------|---------|
| "회의록을 잘 요약해주세요" | "1. 한 줄 요약 작성 2. 결정사항 추출 3. 액션아이템 표로 정리" |
| 애매한 지시 | **구체적인 단계**로 분해 |

> 🎯 **핵심**: Claude가 "이렇게 하면 되겠다"고 따라할 수 있게 쓰세요.

---

## ⚠️ 문제 해결

### "skill-creator가 안 돼요"

```
/plugin install skill-creator@anthropic-agent-skills
```

### "스킬이 자동으로 안 불려요"

1. `/skills`로 설치 확인
2. description에 트리거 키워드 있는지 확인
3. 정확한 키워드 사용 (예: "회의 요약해줘")

### "만든 스킬 위치를 모르겠어요"

- 기본 위치: `.claude/skills/스킬이름/SKILL.md`
- VSCode 파일 탐색기에서 `.claude` 폴더 찾기
