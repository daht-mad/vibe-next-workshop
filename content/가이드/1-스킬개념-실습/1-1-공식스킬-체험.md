## 실습 1: 스킬 수동 설치하기 (폴더 이해용)

> ⭐ 스킬이 어디에 저장되는지 직접 눈으로 확인해봅시다!

### Step 1: GitHub에서 스킬 다운로드

1. [github.com/anthropics/skills](https://github.com/anthropics/skills) 접속
2. 녹색 **Code** 버튼 클릭 → **Download ZIP** 클릭
3. 압축 해제

### Step 2: 스킬 폴더 구조 확인

압축 해제하면 이런 구조가 보입니다:

```
skills-main/
├── skills/
│   ├── pdf/           ← PDF 스킬
│   │   └── SKILL.md
│   ├── pptx/          ← 프레젠테이션 스킬
│   │   └── SKILL.md
│   ├── docx/          ← Word 스킬
│   │   └── SKILL.md
│   └── ...
└── README.md
```

> 💡 **발견**: 스킬 하나 = 폴더 하나 + `SKILL.md` 파일!

### Step 3: 전역 스킬 폴더에 복사

**Mac 터미널**에서:

```bash
# pdf 스킬을 전역 폴더에 복사
cp -r ~/Downloads/skills-main/skills/pdf ~/.claude/skills/
```

### Step 4: 설치 확인

```bash
ls ~/.claude/skills/
```

→ `pdf` 폴더가 보이면 성공!

## 💡 더 쉬운 방법: 명령어 한 줄!

방금 수동으로 한 걸... **명령어 한 줄**로 할 수 있습니다.

### 플러그인이란?

| 개념 | 설명 |
|------|------|
| **플러그인** | 스킬 여러 개를 묶은 **배포 패키지** |
| **마켓플레이스** | 플러그인을 배포하는 **저장소** (GitHub 등) |

> 💡 **비유**: 플러그인 = 앱스토어에서 받는 앱 묶음

### 명령어로 설치하기

**Step 1**: 마켓플레이스 등록 (최초 1회)
```
/plugin marketplace add anthropics/skills
```

**Step 2**: 스킬 설치
```
/plugin install document-skills@anthropic-agent-skills
```

**Step 3**: 설치 확인
```
/skills
```

> 💡 **결과는 같아요!** 명령어가 GitHub에서 다운로드 → `~/.claude/skills/`에 복사를 자동으로 해줍니다.

### 명령어 해석

```
/plugin install document-skills @ anthropic-agent-skills
              ↑                    ↑
          플러그인 이름         마켓플레이스 이름
```

> 💡 **팁**: Claude.ai 유료 플랜 사용자는 이미 설치되어 있어요!

## 실습 2: 스킬 사용해보기

### PDF 생성 체험

**Step 1**: 간단한 내용으로 PDF 만들기
```
오늘 워크샵에서 배운 내용을 정리해서 PDF로 만들어줘:
- 스킬이란 무엇인가
- MCP와 스킬의 차이
- 스킬의 장점
```

**Step 2**: 결과 확인
- PDF 파일 생성됨
- 다운로드해서 열어보기

**Step 3**: 수정 요청해보기
```
제목을 "클로드코드 워크샵 Day 1"로 바꿔줘
```

```
1장: 스킬이란?
2장: 스킬의 장점
3장: 오늘 배운 것
```

## 스킬이 작동하는 원리

방금 체험한 것을 복기해봅시다:

```
1. 사용자: "PDF로 만들어줘"
   ↓
2. Claude: (description 확인) "아, pdf 스킬 쓰면 되겠다"
   ↓
3. Claude: (SKILL.md 읽음) 단계별 절차 확인
   ↓
4. Claude: 절차대로 실행 → PDF 생성!
```

**핵심**: `description`에 "PDF", "만들어줘" 같은 키워드가 있어서 **자동 인식**!

## ⚠️ 문제 해결

### "스킬이 안 보여요"

1. `/skills` 명령어로 설치 확인
2. 설치 안 됐으면: `/plugin install document-skills@anthropic-agent-skills`
3. Claude Code 재시작

### "스킬이 작동 안 해요"

1. 정확한 키워드 사용 (예: "PDF로 만들어줘")
2. 파일이 있는 폴더에서 실행
3. 강사에게 질문!
